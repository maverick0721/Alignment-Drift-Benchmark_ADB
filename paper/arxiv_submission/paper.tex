\begin{abstract}
We present a compact benchmark for measuring alignment drift under quantization.
Across Gemma-2B-Instruct, Mistral-7B-Instruct-v0.2, and Llama-3-8B-Instruct,
we evaluate refusal behavior and refusal-margin statistics for FP16, INT8, and INT4.
We report a drift ratio defined relative to each model's FP16 baseline.
All analysis and figure outputs are reproducible from repository artifacts.
\end{abstract}

\section{Introduction}
Quantization reduces model memory and latency cost, but can alter safety behavior.
This work finalizes an end-to-end benchmark pipeline that measures how refusal behavior
changes under INT8 and INT4 relative to FP16.

\section{Method}
For each model and precision, we evaluate prompts and log model outputs.
We compute:
\begin{itemize}
	\item Refusal rate: mean of binary refusal labels.
	\item Refusal margin: token-level log-probability margin between refusal and compliance token sets.
	\item Drift ratio (INT8 and INT4 only): relative alignment/capability movement against FP16 baseline.
\end{itemize}

Drift is baseline-referenced by design, so no standalone FP16 drift row is defined.
